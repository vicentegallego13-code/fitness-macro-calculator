# Fitness Macro Calculator
This is my first Python project, developed at age 17 as part of my journey into Software Engineering.

## Features
- Calculates maintenance calories and macros (Protein, Fats, Carbs).
- Supports different fitness goals: "Bulk" (surplus) and "Cut" (deficit).
- Implements error handling with try/except blocks to manage invalid inputs.
- Automatically generates a detailed .txt report with the results.

## Technical Skills Applied
- Python logic and modular functions (def, return).
- Clean code and English naming conventions.
- Input normalization (.lower(), .upper()).
- File management and data persistence (with open).
